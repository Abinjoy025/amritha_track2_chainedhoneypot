## What Has Been Built

### Phase 1 — Honeypot Trap

**[app.py](app.py)** — Flask server (port 5000)
- Fake admin login portal capturing IP, credentials, user-agent, headers, payload hash
- Trap endpoints: `/admin`, `/wp-admin`, `/phpmyadmin`, `/api/v1/auth`
- MITRE ATT&CK technique auto-mapping from payload signatures
- Logs all attempts to `logs/honeypot_logs.json`

### Phase 2 — AI Detection Pipeline

**[ai_module/](ai_module/)**
- `train_model.py` — XGBoost classifier trained on CIC-IDS 2017 (78 features, 9 attack classes)
  - Balanced sample weighting for rare classes (Heartbleed, Infiltration)
  - 600 estimators, depth 10, learning rate 0.08
- `predictor.py` — Real-time XGBoost inference with SHAP top-3 feature explanations
- `behavioral_fingerprint.py` — Per-IP attacker profiling with cross-session fingerprint correlation
- `campaign_tracker.py` — Groups attacks into campaigns by TTPs and behavioral hash
- `online_learner.py` — Incremental Random Forest retraining (triggers every 20 new samples)

**[pipeline/](pipeline/)**
- `feature_extractor.py` — Zeek conn.log → 78 CIC-IDS feature vector
- `traffic_capture.py` — ZeekCapture connector

### Phase 3 — OSINT + SOAR

**[ai_module/ip_intelligence.py](ai_module/ip_intelligence.py)**
- ip-api.com geo/ASN (no key), Tor exit list, FireHOL Level-1, ipsum, Emerging Threats
- Optional: AbuseIPDB, VirusTotal, Shodan (via `.env` keys)

**[soar/](soar/)**
- `osint_enricher.py` — AbuseIPDB + Shodan enrichment
- `soar_playbook.py` — Auto IP block (iptables) + container recycle on confidence >= 90%

### Phase 4 — Decentralized Evidence Storage

**[blockchain/](blockchain/)**
- `ipfs_manager.py` — Pinata JWT upload with local `data/ipfs_fallback/` fallback
- `blockchain_manager.py` — Ethereum Sepolia (web3.py 6.x)
- `AttackEvidenceStorage.sol` — Solidity ^0.8.20 smart contract
  - Stores: attackerIP, capturedAt, osintScore, attackType, ipfsCID, reporter
  - Functions: storeRecord, getRecord, getRecordsByIP, getLatestRecords

### Phase 5 — Real-Time Dashboards

**[api/main.py](api/main.py)** — FastAPI backend (port 8000)
- REST endpoints + WebSocket `/ws/live` for real-time event push
- Serves built React frontend

**[frontend/honeyport-ui/](frontend/honeyport-ui/)** — React dashboard
- `LiveFeed.jsx` — Color-coded real-time attack feed with OSINT badges
- `ShapPanel.jsx` — SHAP bar chart + confidence + IPFS CID link
- `Web3Ledger.jsx` — Blockchain records viewer (ethers.js + Sepolia Etherscan links)
- `StatsBar.jsx` — Aggregate statistics header

**[dashboard.py](dashboard.py)** — Legacy Flask dashboard (port 8080)
- Campaigns, behavioral profiles, evidence browsing, IPFS link verification

## Quick Start

```bash
# 1. Activate environment
source venv/bin/activate

# 2. Train model (one-time, requires CIC-IDS CSVs in data/cic-ids/)
python3 ai_module/dataset_downloader.py
python3 ai_module/train_model.py --data data/cic-ids/

# 3. Configure .env
cp .env.example .env
# Edit .env with your Pinata JWT, RPC URL, private key

# 4. Launch all services
python3 start.py
```

Access points:
- Honeypot trap: http://localhost:5000
- React dashboard: http://localhost:8000
- Flask dashboard: http://localhost:8080
